# Steps customization

## Overview

SRE.ai offers eighteen Step types:

* Create Change
* Create Collection
* Add to Collection
* Remove from Collection
* Import Changes
* Commit
* Promote
* Deploy
* Install Package
* Create Package Version
* Create Scratch Org
* Update Jira Issue
* Create Pull Request
* Review Pull Request
* Pull Request Exception
* Summarize Change
* List Components
* Post a Slack Message
